HTML Lenguaje de marcado de hipertexto

  • Es un lenguaje de mercados o de marcas,compuesto por etiquetas
  • Crea estructura completa de página web desde la cabezera hasta pie de página

CC3 Hojas de estilo en cascada

  • Lenguaje de diseño, su objetivo es dar estilo a la web
  • Se emplea para dar colores, formas, tipo de letra a las interfaces
